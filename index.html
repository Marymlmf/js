Linguagem JavaScript

Características

  • Linguagem de programação;
  • Linguagem interpretada;
  • Originalmente foi criado como uma Linguagem front-end, ou seja ele era interpretado e executado no cliente/navegador;
  • Atualmente ele também pode ser executado no backend através do NodeJS;
  • É case-sensitive
  • Francamente tipada,pode sser tipada se programa com TypeScript
  • Sintaxe similar a linguagem C, Java
  • Orientada a objeto, Estruturada
